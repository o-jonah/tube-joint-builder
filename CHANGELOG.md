# Changelog

All notable changes to this project are documented in this file.

## v0.1.0 - 2025-11-20

- Initial release: implemented core 3D tube visualization and editing features.
- Tube creation (rectangular & square), parameter controls, and real-time updates.
- 3D manipulation (position/rotation), rotation snapping (45°/90°), and selection highlighting.
- View modes: wireframe and solid rendering, and basic undo/redo support.
- Prototype joint detection and preview behavior.
- Rendering stack using Three.js + React Three Fiber (`@react-three/drei`).
- Design system with Tailwind CSS and `shadcn/ui` components.
- Tooling: TypeScript with `@` path aliases, Vite configured with `@vitejs/plugin-react-swc`.
- Documentation: updated `README.md` with setup and completed features.
